# Сервис инфраструктурной доступности

Веб-сервис для оценки и визуализации пространственной доступности объектов социальной инфраструктуры на территориях с низкой плотностью населения.

https://github.com/user-attachments/assets/dd85ae55-8aa9-4e9e-b548-6f70356c16e0
---

## Что умеет сервис

- **Анализ доступности** — проверяет соответствие нормативам СанПиН для выбранного жилого дома:
  - детский сад в 500 м пешком
  - школа в 500 м пешком или 15 мин на транспорте
  - больница в 2 км пешком или 30 мин на транспорте
- **Изохроны** — строит зоны пешей и транспортной доступности и отображает их на карте
- **Предложения размещения** — при нарушении норматива предлагает оптимальные точки для строительства нового объекта с учётом плотности жилой застройки и дорожной сети
- **Загрузка OSM-данных** — для любой точки России автоматически подтягивает актуальные данные из OpenStreetMap и сохраняет в PostGIS
- **Выбор точки** — кликом по дому на карте, вводом координат вручную или указанием произвольной точки в режиме crosshair

## Стек

| Слой | Технологии |
|------|-----------|
| Frontend | React 18, TypeScript, Vite, OpenLayers 10 |
| Backend | Python 3.12, FastAPI, GeoPandas, Shapely, NetworkX, OSMnx 2.1 |
| База данных | PostgreSQL 16 + PostGIS 3.4 |
| Контейнеры | Docker, Docker Compose |

## Запуск

```bash
# Только БД (для локальной разработки)
docker-compose up db -d
cd backend && python -m db.init_db

# Бэкенд
cd backend && uvicorn main:app --reload --port 8000

# Фронтенд
cd frontend && npm run dev
```

```bash
# Всё через Docker
docker-compose up --build
docker-compose exec backend python -m db.init_db
```

Swagger UI: http://localhost:8000/docs  
Приложение: http://localhost:5173
